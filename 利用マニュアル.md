# コンテンツ制作管理アプリ 利用マニュアル

社内でコンテンツ（Instagram・X・イベント等）の制作進捗を管理するためのツールです。
URLにアクセスするだけで、誰でもブラウザから使えます。

---

## 1. 画面の見かた

トップページにプロジェクトが「カード」で並びます。

- **進行中 / 完了 / すべて** … 上のタブで切り替え
- 各カードに表示されるもの：
  - 種別（Instagram・X・イベント・LINE誘導 など）
  - タイトル
  - コード（担当者・案件コード）
  - 納期
  - 進捗バー（◯/◯ ステップ完了）

---

## 2. 探す・絞り込む

検索バーとボタンで目的のプロジェクトをすぐ見つけられます。

| 操作 | 場所 |
|---|---|
| キーワード検索 | 上の「タイトル・コードで検索」欄 |
| 種別で絞る | Instagram / X / イベント などのボタン |
| コードで絞る | 「コード:」の行のボタン（forest_x など） |
| 並び替え | 登録順 / 締切順 / コード順 |
| 表示切替 | 制作管理 / スケジュール |

### 自分の担当だけ見たいとき
「コード:」の行で自分のコードを押すと、その案件だけ表示されます。
さらに「🔗 リンクをコピー」を押すと、**そのコードだけ表示される専用URL**が作れます。
チームメンバーにそのURLを渡すと、最初からその人の案件だけ見える状態になります。

---

## 3. ステップ（作業工程）を進める

カードの **「ステップを表示」** を押すと、各工程が一覧で出ます。

### ステータスを変える
各ステップの右側のメニューから状態を選びます：

| ステータス | 意味 |
|---|---|
| 未着手 | まだ手をつけていない |
| 素材待ち | クライアント/外注からの素材待ち |
| 素材受領 | 素材が届いた |
| 進行中 | 制作作業中 |
| 確認待ち | 確認・チェック待ち |
| 完了 | このステップ終了 |
| ロック中 | 前の工程が終わるまで着手できない |

> ステータスを変えると即座に反映され、進捗バーも自動更新されます。
> （設定によってはChatwork/Discordに通知が飛びます）

### ロック（鍵）と工程の紐付け
ステップ同士に「順番」を設定できます。あるステップを別のステップに**紐付ける**と、紐付け先が「完了」になるまで、そのステップは**ロック中**（着手不可）になります。

- 例：「デザイン制作」を「素材受領」に紐付けると、素材が揃うまでデザインはロック中
- 紐付け先が完了すると、ロックは自動で解除され「未着手」になります
- 前の工程が終わっていないのに先に進むミスを防げます

### 素材・ファイルを提出する
ステップにURLやファイル、メモを添付できます。デザインデータや参考リンクの共有に使えます。

---

## 4. 新しいプロジェクトを作る

### 方法A：1件ずつ作る
右上の **「＋ 新規プロジェクト」** から作成します。
※このボタンは管理者用の合言葉URLが必要です（管理者に確認してください）。

入力するもの：
- タイトル
- 制作種別（既存から選ぶ or その場で新種別を追加）
- プロジェクトコード（担当者・案件コード）
- 納期
- 説明（任意）

### 方法B：CSVでまとめて作る
たくさんのプロジェクトを一度に作るときに便利です。

1. 右上の **「CSV取込」** を押す
2. 「サンプルCSV」をダウンロードして形式を確認
3. 表計算ソフト（Excel・スプレッドシート）で作成 → CSVで保存
4. ファイルをドロップ → プレビュー確認 → 「一括作成」

**CSVの形式：**
```
タイトル,種別,コード,納期,ステップ名,担当,ステップ期日,備考
```
- 同じタイトルの行は、1つのプロジェクトのステップとしてまとまります
- **同じタイトルが既にある場合は上書き**されます（最新版に更新）
- 「担当」は登録済みの役割名（クライアント/外注/自社 など）を入れるとマッチします

---

## 5. 表示の切り替え（制作管理 / スケジュール）

右上のボタンで2つの表示を切り替えられます。

- **制作管理**：プロジェクトをカードで並べて進捗・ステップを管理する通常画面
- **スケジュール**：納期やステップ期日をカレンダー形式で確認できる画面

---

## 6. 設定（右上の ⚙️ ボタン）

歯車ボタンを押すと、3つのタブで設定できます。

### 役割タブ
- 役割（クライアント・外注・自社 など）の名前と色を変更
- 「役割を追加」で新しい役割を追加
- 不要な役割は削除
- 上下の矢印で並び替え

### ステータスタブ
- ステータス（未着手・進行中・完了 など）の名前・色を変更・追加・削除
- 名前を変えると既存ステップにも反映
- 並び替え可能

### 通知タブ
ステータス更新や新規作成時に、ChatworkまたはDiscordへ自動通知を送れます。

- 「通知なし / Chatwork / Discord」から送り先を選択
- **Chatworkの場合**：APIトークンと、ルームの番号（ルームID）を入力
- **Discordの場合**：チャンネルで作成したWebhook URLを貼り付け
- 「テスト通知を送る」ボタンで届くか確認できます

> Chatworkのルームidは通知したいチャットのURL末尾の数字。
> DiscordのWebhook URLはチャンネル設定→連携サービス→ウェブフックで作成します。

## 7. その他の操作

| やりたいこと | 操作 |
|---|---|
| プロジェクトを複製 | カード右上のコピーアイコン |
| プロジェクトを削除 | カード右上のゴミ箱アイコン（即消える） |
| タイトルを編集 | タイトル横の鉛筆アイコン |

## 8. よくある質問

**Q. 編集した内容は自動で保存される？**
A. はい。ステータス変更・編集はすべて自動保存されます。保存ボタンは不要です。

**Q. 他の人が変更したら自分の画面にも反映される？**
A. はい。リアルタイムで反映されます（ページを開いたままでOK）。

**Q. 間違えて削除した**
A. 管理者に連絡してください。削除はすぐ画面から消えますが、データは復元できる場合があります。

**Q. 役割名（クライアント名など）を変えたい**
A. 右上の ⚙️ 設定 →「役割」タブで変更できます。

---

困ったときは管理者（システム担当）に連絡してください。
